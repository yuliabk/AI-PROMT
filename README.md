# AI-PROMT-BILDER

Static GitHub Pages version of AI Prompt Builder.

## What this version includes

- `index.html` - the full interactive app.
- `404.html` - duplicate page for GitHub Pages fallback.
- `.nojekyll` - prevents GitHub Pages processing issues.
- No React build required.
- No API key required.
- Works directly in the browser.

## GitHub Pages setup

1. Create a public GitHub repository.
2. Upload the files from this folder directly to the root of the repository.
3. Go to Settings → Pages.
4. Choose Deploy from a branch.
5. Branch: main.
6. Folder: /root.
7. Save.
8. Open the published link.

If the repository is named `AI-PROMT-BILDER`, the URL will usually be:

```text
https://yuliabk.github.io/AI-PROMT-BILDER/
```
